Estudo Estrutura de Dados

Linguagens

Caracteristicas

O que eh uma linguagem?

  • Forma de descrever um processo
  • Possui uma estrutura e conjunto de regras a ser usado nesta descrição
  • Pode ser caracterizado por:
    • Nivel de abstração
    • Objetivo
    • Forma de execução
    • Paradigma

Nível de Abstração

Abstração

  • Quando programamos, estamos orientando a máquina a fazer algo que desejamos
  • O modo de informar a máquina sobre o que pode fazer pode ser mais direto ou mais abstrato
  • Exemplo:
    • Assembly (direto, baixo nível)
    • SQL (Abstrato, muito alo nível, declarativo)

Para que serve a linguagem

Objetivo

  • Algumas linguagens são contruídas para objetivos específicos, outras são mais genéricas.
  • Exemplo:
    • Prolog (objetivo IA)
    • C (genérica)

Modo como executa

Forma de execução

  • Temos linguagens interpretadas e compiladas
  • Cada tipo de linguagem possui caraterísticas que tornam o fato de ela ser compilada ou interpretada mais adequada
  • Exemplo:
    • Python (interpretada)
    • C (compilada)
